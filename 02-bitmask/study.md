1. 비트연산자 우선순위
    - &(and), |(or), ^(xor) 등은 비교연산자(==, !=)보다 우선순위가 낮다!! 
    ```
    int c = (6 & 4 == 4)
    ```
    - 위 코드는 4 == 4 먼저 계산됨. 즉, 6 & 1이라 0 리턴함. 
        => 괄호를 자세하게 쓰자 !! 

2. 비트마스크를 집합으로..! 
    - 집합 {1, 4, 5, 6, 7, 9}를 비트로 표현하려면 -> 10 1111 0010 으로 표현가능하다. 
    즉, 2^1 + 2^4 + 2^5 + 2^6 + 2^7 + 2^9 = 754로 표현됨. 
    - 1. 공집합과 꽉 찬 집합 표현 
    ```
    int fullPizza = (1 << 20) - 1
    ```
    1을 왼쪽으로 20칸 이동시키면, 21번째에 1이 있고, 뒤엔 20개의 0이 존재함. 이걸 - 1 해주면 20개 모두가 1로 켜짐. 
    공징합은 그냥 0 !
